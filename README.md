# 濮阳同城宠物交易网站 Puyang Pet Market
## 项目介绍
面向濮阳市本地的宠物交易、免费领养、配种、寄养信息发布平台
覆盖区域：华龙区、濮阳县、清丰县、南乐县、范县、台前县
纯本地自提交易，规避线上异地诈骗，适合濮阳养宠人群使用。

## 技术栈
前端：HTML5 + CSS3 + JavaScript
后端：Node.js + Express
存储：JSON文件（无需安装MySQL，轻量化部署）
图片上传：Multer

1. 安装Node.js（官网下载）
2. 克隆/下载本项目到本地
3. 打开终端进入项目根目录，执行：
